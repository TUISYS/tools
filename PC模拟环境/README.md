## VC++模拟环境概述：
微软提供的vc++开发环境，可以模拟TUI的应用开发，在`platform/vc++`文件夹下面，可以打开`xxx.vcxproj`工程文件，编写自己的应用逻辑。只包含`tui.h`头文件的`.c`，各个平台之间的代码是可以相互兼容的。
